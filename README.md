# 🍞 100白노트로 암기빵!

## 1. 프로젝트 소개
'100白노트로 암기빵!'은 '백지노트 암기 방식'을 디지털로 구현한 자가 학습용 웹 서비스다. 이 사이트를 통해 사용자가 직접 학습할 키워드와 내용을 입력하여 나만의 암기장을 만들고, 빈칸 채우기 퀴즈를 통해 학습 상태를 점검할 수 있다. 복잡한 외부 앱 설치 없이 웹 브라우저만으로 가볍고 빠르게 암기 상태를 테스트할 수 있도록 기획했다.

## 2. 주요 기능
* **암기장 다중 관리 (로컬 스토리지 적용):** 여러 과목이나 단원(예: 중간고사, 기말고사 등)을 나누어 관리할 수 있도록 좌측 사이드바에 문서 리스트 기능을 구현했다. 사용자가 입력한 데이터는 브라우저의 로컬 스토리지에 자동 저장되어, 새로고침을 해도 데이터가 날아가지 않는다.
* **문제 출제 모드:** 문제 제목, 정답 내용, 그리고 채점의 기준이 될 '필수 포함 키워드'를 사용자가 직접 쉼표로 구분하여 입력할 수 있다.
* **시험 보기 및 유사 채점 모드:** 시험 모드로 전환하면 정답 내용이 가려진 카드가 출력된다. 사용자가 답안을 작성하여 제출하면, JavaScript가 필수 키워드의 포함 여부를 계산하여 정답률을 도출한다.
* **시각적 피드백 (암기빵 UI):** 딱딱한 점수 대신, 정답률에 따라 빵이 갉아먹히는 듯한 이모지 변화(🍞 ➔ 🥐 ➔ 🥯 ➔ 🤤)를 주어 학습의 흥미를 높였다. 채점 후에는 원래 정답 텍스트 안에서 맞춘 키워드만 시각적으로 하이라이트 처리하여 보여준다.

## 3. 개발 및 구현 방식 (바이브코딩)
본 프로젝트는 AI 에이전트와 협업하는 '바이브코딩' 방식을 통해 구현되었다.
1. **문제 분해 및 PRD 작성:** 5W1H 프레임워크를 기반으로 핵심 기능을 '데이터 입력 ➔ 암기 퀴즈 생성 ➔ 유사 채점 로직 ➔ 시각화'로 잘게 분해하여 명확한 프롬프트를 작성했다.
2. **보안 및 효율성 고려:** 외부 AI API 연동 시 발생할 수 있는 보안 문제(API Key 노출)와 할루시네이션을 방지하기 위해, 순수 HTML, CSS, Vanilla JavaScript만을 사용하여 클라이언트 사이드에서 안전하게 작동하도록 타협점(키워드 매칭 유사 채점)을 찾아 구현했다.
